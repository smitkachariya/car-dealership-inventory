# car-dealership-inventory

A full-stack Car Dealership Inventory API and SPA built using strict Test-Driven Development (TDD) for the Incubyte assessment.

## Features

- User registration and login
- JWT authentication
- Vehicle inventory management
- Vehicle search and filtering
- Purchase vehicles
- Restock inventory
- Role-based access control

## Tech Stack

### Frontend

- React
- Vite
- Axios

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
